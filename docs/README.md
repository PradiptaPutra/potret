# Screenshots

Drop screenshots here to make them show up in the main README. Capture them with
Potret itself for the best look, then save with these exact names:

| File | Suggested shot |
|------|----------------|
| `quick-access.png` | The floating Quick Access popup sitting over a fresh capture |
| `annotate.png` | The annotation editor with an arrow / box / text drawn |
| `background.png` | The Background tool with a gradient + padding + shadow |

Tip: a clean desktop and a real app/website behind the capture makes the shots pop.
Then commit the PNGs and the README images will render automatically.
